安装jenkins插件

1.安装jenkins插件可以使用在线、离线方式：

  • 在线：Jenkins－系统管理－管理插件－可选插件－直接安装
  • 离线：Jenkins－系统管理－管理插件－高级－上传插件

2.安装以下插件（按需选择即可）：

  • Checkstyle Plug-in
  • Ant Plugin
  • Maven Info Plugin
  • Maven Repository Server Plugin
  • GitLab Plugin
  • Gitlab Hook Plugin
