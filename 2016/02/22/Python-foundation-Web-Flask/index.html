常见的Python Web框架

  • Flask：一个轻量级Web应用框架。
  • Django：一个全能型Web框架。
  • Tornado：Facebook的开源异步Web框架。
  • web.py：一个小巧的Web框架。
  • web2py：Google在web.py基础上二次开发而来的。
  • Bottle：Bottle和Flask都属于轻量级的Web框架。

Flask是一个使用 Python 编写的轻量级 Web 应用框架。Flask 依赖着两个外部库：Jinja2 模板引擎以及 Werkzeug WSGI 工具集。
