Eclipse安装CheckStyle

1.下载CheckStyle插件

在网页Eclipse Checkstyle Plugin中点击Download，下载插件压缩包到本地。

2.点击Help->Install New Software，按照页面提示操作即可，如下图所示。

3.点击Archive导入已下载的压缩包。

4.如果页面停留在“Calculating requirements and dependencies.”，则断开网络（拔掉网线~），即可解决问题。

Eclipse的CheckStyle配置

2.点击 Project->Properties->CheckStyle

Eclipse配置tabs为spaces

1.点击 Eclipse->Preference

依次选择 General->Editors->Text Editors，选中右侧的 Insert spaces for tabs；保存。

2.点击 Eclipse->Preference

依次选择 Java->Code Style->Formatter，点击右侧的Edit，选则左侧Tab policy的值为Spaces only，保存。

3.错误解决办法

错误提示：“This is a built-in profile, change the name to create a new profile.”。返回上一级，点击New按钮，根据当前的样式重新生成一个新的样式并保存，重复上一步即可。
